# BeyondChats Email Dashboard - Frontend

This is the React frontend application for the BeyondChats Email Dashboard assignment. 

## Features
- Dynamic Inbox & Sent Folders
- Interaction Analytics Dashboard
- Smart Content Filters & Colorful Topic Tagging
- LocalStorage based Star/Bookmark System
- Pro Keyboard Shortcuts (`/` for search, `j`/`k` for navigation, `r` for reply)
- Responsive Tailwind CSS Design
- OAuth 2.0 Integration Client

## Setup
All primary documentation and setup instructions are located in the [Root README](../README.md).

To run this frontend locally:
1. Ensure the backend API is running on port 8000.
2. Run `npm install`
3. Run `npm start`

The application will launch at `http://localhost:3000`.
